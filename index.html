Hola yo soy fabian y estoy aprendiendo a usar Git

git init---solo se hace al iniciar un nuevo proyecto hace que Git empieze a monitoriar todo lo que hagamos

git status--sirve para ver el estado del proyecto

git add--.sirve para agregar un archivo al commit si ya trabajamos en nuestros documentos

git add -A--sirve para agregar todo los archivos

git commit -m "Msg"--sirve para guardar los cambios que se hiceron, el msj sirve para identificar que modificamos agregar commit
